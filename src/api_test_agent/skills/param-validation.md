# 参数验证测试策略

## 必填与选填
- 必填参数不传 → 应返回 400，错误提示包含字段名
- 必填参数传空值（""、null）→ 应返回 400
- 选填参数不传 → 应正常工作
- 只传必填参数 → 应正常工作

## 数据类型
- 期望 int 传 string → 应返回 400
- 期望 string 传 number → 视接口而定
- 期望 boolean 传 "true"（字符串）→ 视接口而定

## 边界值
- 数值型：最小值、最大值、最小值-1、最大值+1、0、负数
- 字符串：空串、1个字符、最大长度、超过最大长度
- 数组：空数组、1个元素、最大数量、超过最大数量

## 特殊字符
- SQL 注入：' OR 1=1 --
- XSS：<script>alert(1)</script>
- 超长字符串
- Unicode/emoji
